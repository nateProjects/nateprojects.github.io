JavaScript Course

Nate

Since the age of fourteen I've been programming. It's never been my full time job, but I have been able to use the coding skills I've learnt often in different technical jobs I've held.

From the early days of the web I've been making web pages using HTML, CSS, and JavaScript. But, after I completed the Code Academy & Code School lessons for that language, I found that both those courses were lacking in one area: teaching how to change web page elements natively (without any special libraries like JQuery). So I decided to create my own little course to help those interested.

In the past I've done video training for the log monitoring application Splunk too.
